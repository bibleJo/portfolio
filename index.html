About

I'm Jo sung kyoung

안녕하세요, 저는 강한 책임감과 열정을 지닌 신입 퍼블리셔 조성경 입니다. 저는 퍼블리셔라는 직업에 흥미를 느끼고 새로운 시작을 하게 되어 따라잡기가 아닌 전보다 나은 사이트 구축을 선구합니다. 사용자 입장에서 쉽게 접근할 수 있는 웹 구축을 위해 팀원들과 소통하며 구조화된 마크업을 목표로 하고 있습니다. 저에게 코딩이란 부지런하게 삶을 살 수 있게 하는 원동력과도 같으며 잘하고 싶어 노력하는 일이기도 합니다. 더 한발 나아가기 위해 끊임없이 노력하며 늘 배우고 탐구하는 자세로 소통과 기획능력까지 갖춘 웹퍼블리셔가 되겠습니다.

Skill

  • HTML

    웹 표준과 웹 접근성을 준수하는 HTML 정보 구조설계로 웹사이트 퍼블리싱을 해오고 있습니다. 특히 웹 표준을 위해 HTML5에서 권장하는 마크업 방식과 시멘틱 태그를 사용해 전체 레이아웃을 설계합니다. 웹 UI 디자인 작업 후 클래스 네임과 함께 HTML 구조설계를 꼼꼼히 작업한 후 퍼블리싱 작업을 진행합니다.

  • CSS

    CSS 코딩은 IE를 지원도 충분히 고려하는 크로스브라우징(Cross Browsing)을 위해 안정적인 CSS속성을 사용해서 HTML Elements를 배치하고 디자인합니다. 특별한 경우 CSS3에서 새롭게 지원하는 속성을 활용해서 좀 더 다양한 CSS 배치와 디자인을 합니다.

  • JQUERY

    네비게이션, 탭 메뉴, 아코디언 등 웹 퍼블리싱에 반드시 필요한 상호작용을 위한 제이쿼리는 플러그인을 사용하지 않고 순수한 제이쿼리 코딩으로 제작합니다. 필수적인 자바스크립트 활용은 가능하며 주로 제이쿼리와 병행해서 활용합니다.(if 조건문, for 반복문 등)

  • RESPONSIVE WEB

    반응형 웹 제작에 있어서 크로스브라우징(Cross Browsing)을 위해 안정적인 기존 방식인 float, position, Media Query 를 활용한 반응형 웹 제작 경험을 충분히 가지고 있습니다. 필요에 따라 플렉스(Flex)와 그리드(Grid) 방식으로 반응형 웹 사이트 제작도 충분히 가능합니다. 대부분 기존 방식으로 제작하지만 모바일 웹 앱 제작의 경우 Flex와 Grid를 충분히 활용해서 제작합니다.

  • MOBILE WEB APP

    모바일 우선을 위해 Mobile First에 준하는 모바일 웹 앱 제작 경험도 충분히 가지고 있습니다. UI/UX 측면에서 미적인 디자인 뿐만 아니라 모바일 사용자 경험에 최적화를 위해 효율적인 컨텐츠 배치를 우선적으로 생각합니다. 그리고 모바일 대응을 위한 플랫폼으로 반응형 웹, 모바일 웹, 모바일 웹 앱, 네이티브 앱에 대한 충분한 이해도를 갖고 있습니다.

  • TEXT Editor

    웹 퍼블리셔로서 텍스트에디터는 Visual Studio Code를 메인으로 사용하며 경우에 따라 Adobe Brackeets를 병행해서 사용합니다. 간단한 부분 수정을 위해 EditPlus, NotePad++ 도 활용합니다.
